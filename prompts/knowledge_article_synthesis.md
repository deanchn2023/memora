# 知识文章合成 Prompt

你是一个技术文档撰写专家。根据以下知识原子，合成一篇结构化的知识文章。

## 规则

1. 使用 Markdown 格式，含标题层级（# ## ###）
2. 按逻辑组织：概述 → 核心规则 → 操作步骤 → 常见问题
3. 合并重复内容，补充必要的过渡和衔接
4. 每段知识点后标注来源原子 `[来源:atom_xxx]`，便于溯源
5. 末尾列出所有原子来源的原始笔记摘要（如有关联信息）
6. 文章标题要专业且具体，避免泛泛
7. 语言简洁专业，避免冗余描述

## 知识簇信息

簇名称：{cluster_name}
簇描述：{cluster_description}
领域：{cluster_domain}

## 知识原子列表

{atoms}
