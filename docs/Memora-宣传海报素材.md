## 核心优势（6 大卖点）

### 1. 零感录入 — 复制即识别
复制任何文字，3 秒内 AI 自动完成三重判断：
- **待办提取**："@Dean 你整理一下方案" → 自动创建高优先级任务
- **信息归档**：产品规格参数 → 自动分类保存到记事本
- **知识推荐**：疑问语句 → 自动触发语义搜索推送相关内容
- 无效内容（闲聊/确认语）静默丢弃，零打扰

### 2. 知识自动沉淀 — 碎片变体系
三阶段知识萃取引擎，无需手动整理笔记：
- **原子提取**：笔记 → 最小知识单元（事实/规则/洞察/方法/问题）
- **智能聚类**：AI 按主题自动分组，优先归入已有簇
- **文章合成**：成熟簇（≥5 原子）→ 结构化 Markdown 文章，可溯源

### 3. 记忆自演进 — 越用越精准
三层记忆自动分层 + AI 反馈闭环：
- **记忆分层**：瞬时 → 短期 → 长期，AI 根据语义自动判定层级
- **反馈学习**：你删过的 AI 不再犯，你改过的 AI 学会你的偏好
- **画像自适应**：高频人物/活跃项目/优先级偏好，AI 越来越懂你

### 4. 五专家 Agent — 一个入口全搞定
对话自动分流到专业 Agent，无需切换工具：
- **Priority Agent**：综合截止时间 + 工作时段，为你排 Top 5
- **Knowledge Agent**：碎片聚类、合并建议、洞察提取
- **Memory Agent**：记忆晋升/降级/淘汰/合并建议
- **Report Agent**：自动生成日报/周报，含完成情况 + 明日建议
- **Chat Agent**：上下文感知通用对话（注入你的记忆/任务/笔记）

### 5. 本地优先 — 隐私零妥协
数据在本地，智能在云端：
- 所有记忆、笔记、知识存储在本地 SQLite + JSON，不上传云端
- AI 调用最小化：CRUD 不依赖 AI，断网基础功能可用
- API Key 环境变量注入，不硬编码
- 反馈数据本地存储，不外传

### 6. 多端生态 — PC 深度操作 + 移动随身查看
- **PC 端（Electron）**：剪贴板感知、知识萃取、深度编辑、报告生成
- **移动端（Flutter）**：待办查看、轻量记事、知识浏览、AI 对话
- **云端枢纽（Config Server）**：数据同步、配置下发、用户鉴权
- 三端实时同步，revision 乐观锁 + 设备注册机制

## 技术亮点

| 维度 | 实现方式 |
|------|---------|
| 桌面框架 | Electron 28（macOS / Windows / Linux） |
| AI 引擎 | DeepSeek（结构化提取）+ 腾讯 ADP V2（语义搜索/对话） |
| 数据存储 | 本地 SQLite + JSON（零云端依赖） |
| Prompt 引擎 | Handlebars 模板 + 动态变量注入 + 在线编辑实时生效 |
| 反馈系统 | trace 日志 + 正/负样本自动注入 few-shot |
| 国际化 | 中/英双语，运行时无缝切换 |
| 设计语言 | Apple Design Language（毛玻璃 + SF Pro + Spring 动效） |

## 数据流一目了然

```
你复制文字 → AI 意图识别（3秒）
    ├→ 待办 → 任务列表（自动优先级 + 关联项目）
    ├→ 信息 → 记事本（自动分类 + 摘要）
    ├→ 疑问 → 知识搜索（ADP语义 + 本地 + 公开资源）
    └→ 无效 → 静默丢弃（零打扰）

你写笔记 → AI 知识萃取（一键）
    → 知识原子 → 智能聚类 → 结构化文章（可溯源）

你对话 → AI 自动路由到专家 Agent
    → RAG 检索你的记忆/任务/笔记 → 精准回答

你操作 → 反馈闭环（自动）
    → 正/负样本 → 注入后续 Prompt → AI 越来越懂你
```
