# TaskFlow - 智能事项管理系统

> 版本：1.0.0
> 最后更新：2026-05-27

---

## 一、项目概述

### 1.1 项目名称
**TaskFlow** - 基于 AI 的智能事项管理系统

### 1.2 核心价值
- **零输入创建**：从剪切板自动识别待办事项
- **智能时间预估**：AI 分析任务所需时间
- **番茄钟管理**：25分钟专注 + 5分钟休息
- **多维视图**：日/周/月三种时间维度
- **智能提醒**：足够时间提醒 + 临近交付提醒

### 1.3 技术栈
| 层级 | 技术 | 版本 |
|------|------|------|
| 框架 | Electron | ^28.0.0 |
| 前端 | HTML + CSS + JavaScript | - |
| 样式 | Apple Design Language | - |
| 存储 | localStorage + IndexedDB | - |
| AI | DeepSeek API | v4-flash |
| 日历 | macOS Calendar (AppleScript) | - |

## 三、数据模型

### 3.1 Task（任务）

```javascript
{
  "id": "task_1703836800000",
  "title": "完成项目报告",
  "description": "整理本周工作内容，生成周报",
  "estimatedDuration": 120,        // 预估时长（分钟）
  "actualDuration": 0,            // 实际时长（分钟）
  "priority": "high",             // high | medium | low
  "status": "pending",            // pending | in_progress | completed | cancelled
  "dueDate": "2026-05-28T17:00:00.000Z",
  "reminderSettings": {
    "enoughTime": 120,            // 足够时间提醒（交付前N分钟）

  },
  "reminders": [
    "2026-05-28T15:00:00.000Z",   // 足够时间提醒

  ],
  "pomodoroSessions": [],         // 番茄钟记录
  "calendarEventId": null,        // 系统日历事件ID
  "source": "clipboard",          // clipboard | manual

```

## 五、核心功能模块

### 5.1 剪切板监听模块

**功能描述**：
- 定时轮询剪切板内容（默认2秒）
- 对比上次内容，检测变化
- 变化时调用 AI 分析是否包含待办事项

**AI 分析 Prompt**：
```
分析以下文本是否包含待办事项或任务。如果是，提取以下信息：
1. 任务标题（简洁明确）
2. 任务描述（详细说明）
3. 截止时间（如果有提及）
4. 优先级（根据紧急程度判断：high/medium/low）
5. 预估所需时间（分钟）

文本：{clipboard_text}

请以 JSON 格式返回，如果不含待办事项则返回 null：
{
  "isTask": true/false,
  "task": {
    "title": "",
    "description": "",
    "dueDate": null 或 "YYYY-MM-DD HH:mm",
    "priority": "high/medium/low",
    "estimatedDuration": 数字
  }
}
```

### 5.2 番茄钟模块

**状态机**：
```
┌─────────┐  开始  ┌─────────┐  完成  ┌─────────┐
│  空闲   │ ────→ │  工作   │ ────→ │ 短休息  │
│         │        │ (25min) │        │ (5min)  │
└─────────┘        └─────────┘        └─────────┘
     ↑                                    │
     │                                    ↓
     │              ┌─────────┐    ┌─────────┐
     └──────────────│ 长休息  │ ←──│ 计数=4  │
                    │ (15min) │    └─────────┘
                    └─────────┘
```

**功能**：
- 25分钟工作时间
- 5分钟短休息
- 每4个番茄钟后15分钟长休息
- 统计今日完成番茄钟数量
- 关联到具体任务

### 5.3 时间视图模块

**日视图**：
- 时间轴从 06:00 到 22:00
- 每30分钟一个格子
- 任务按时间块显示
- 空闲时间显示为空白
- 支持拖拽调整任务时间

**周视图**：
- 显示本周7天
- 每天显示任务数量和预览
- 点击某天跳转到日视图

**月视图**：
- 显示当月日历
- 有任务的日期显示圆点标记
- 显示当天任务数量
- 点击某天跳转到日视图

### 5.4 提醒模块

**双重提醒机制**：

```
任务: 完成项目报告
截止: 2026-05-28 17:00
预估: 2小时

提醒时间线：
────────────────────────────────────────────→
              │                    │        │
           创建时              足够时间    临近提醒
          (立即)              (提前2h)    (提前30min)
                               ↓            ↓
                           15:00提醒    16:30提醒
                           "请预留      "即将到期！"
                            足够时间"
```

**通知内容**：
```javascript
{
  title: "TaskFlow 提醒",
  body: "任务「完成项目报告」将在30分钟后到期",
  icon: "path/to/icon.png",
  sound: "default",
  actions: [
    { type: "button", text: "开始任务" },
    { type: "button", text: "稍后提醒" }
  ]
}
```

### 5.5 日历同步模块

**macOS 日历集成**：
- 使用 AppleScript 操作系统日历
- 创建独立日历 "TaskFlow"
- 任务创建时自动添加日历事件
- 任务更新时同步更新日历事件
- 任务删除时同步删除日历事件

**AppleScript 示例**：
```applescript
tell application "Calendar"
  tell calendar "TaskFlow"
    make new event at end of events with properties ¬
      {summary:"完成项目报告", ¬
       start date:date "2026-05-28 15:00:00", ¬
       end date:date "2026-05-28 17:00:00", ¬
       description:"预估时长: 2小时"}
  end tell
end tell
```

## 六、API 设计

### 6.1 DeepSeek API 调用

**任务分析接口**：
```javascript
async function analyzeTask(text) {
  const response = await fetch('https://api.deepseek.com/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${API_KEY}`
    },
    body: JSON.stringify({
      model: 'deepseek-v4-flash',
      messages: [
        {
          role: 'system',
          content: '你是一个任务分析助手，擅长从文本中提取待办事项信息。'
        },
        {
          role: 'user',
          content: `分析以下文本是否包含待办事项：\n\n${text}`
        }
      ],
      response_format: { type: 'json_object' }
    })
  });
  
  return await response.json();
}
```

**时间预估接口**：
```javascript
async function estimateDuration(task) {
  const response = await fetch('https://api.deepseek.com/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${API_KEY}`
    },
    body: JSON.stringify({
      model: 'deepseek-v4-flash',
      messages: [
        {
          role: 'system',
          content: '你是一个时间管理专家，能够准确预估任务所需时间。'
        },
        {
          role: 'user',
          content: `预估以下任务需要多少分钟：\n\n任务：${task.title}\n描述：${task.description}`
        }
      ]
    })
  });
  
  return await response.json();
}
```

## 十、注意事项

1. **隐私安全**：所有数据本地存储，API Key 仅存储在本地配置文件
2. **剪切板权限**：首次运行需要用户授权剪切板访问权限
3. **日历权限**：首次同步需要用户授权日历访问权限
4. **通知权限**：首次发送通知需要用户授权通知权限
5. **性能优化**：剪切板轮询间隔建议不低于1秒，避免频繁调用 API
6. **错误处理**：API 调用失败时提供友好的错误提示和重试机制
